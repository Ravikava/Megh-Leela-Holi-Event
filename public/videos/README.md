# Video Directory

Place your Holi festival video here as `holi-video.mp4`.

## Video Requirements

- Format: MP4
- Recommended resolution: 1920x1080 (Full HD) or higher
- Recommended codec: H.264
- File size: Keep it optimized for web (under 10MB if possible)

## Fallback

If no video is provided, a colorful gradient background will be displayed automatically.
